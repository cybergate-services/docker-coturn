# Installing  Coturn Docker

**1. Clone the repository
```
# cd /opt
# git clone https://github.com/cybergate-services/docker-coturn.git
# cd /opt/docker-coturn
```
**2. To Setup docker compose envioronment run: 
```
# ./set-env.sh
```
**3. Install the containers
```
# docker-compose up -d
```
